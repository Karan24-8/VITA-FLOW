# 🩺 VITA-FLOW — Your Health, Precisely Planned

🌐 **Live Demo:** https://vita-flow-pi.vercel.app/
💻 **GitHub Repo:** https://github.com/Karan24-8/VITA-FLOW

## 🚀 Overview

**VITA-FLOW** is a full-stack, AI-powered health management platform built as a DBMS group project. It enables users to seamlessly manage their health by tracking meals, logging workouts, and booking expert consultations — all in one unified system.

---

## 🛠️ Tech Stack

* **Backend:** Node.js, Express.js
* **Database:** PostgreSQL (Supabase)
* **Frontend:** Vanilla JavaScript, HTML, CSS
* **AI Integration:** External AI APIs
* **Deployment:** Vercel (Frontend), Render (Backend)

---

## 📦 Modules

### 🍽️ Food

* Log daily meals
* Track macronutrients
* Generate AI-powered diet plans

### 🏋️ Exercise

* Browse exercise database
* Log workout sessions
* Get AI-generated workout routines

### 🧑‍⚕️ Consultant

* Book expert appointments
* AI-generated pre-consult summaries

### 🔐 Authentication

* JWT-based authentication
* Role-Based Access Control (RBAC)

  * User
  * Admin
