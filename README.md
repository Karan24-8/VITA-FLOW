VITA-FLOW — Your Health, Precisely Planned
🌐 Live: https://vita-flow-pi.vercel.app/ | 💻 GitHub: https://github.com/Karan24-8/VITA-FLOW
A full-stack AI-powered health management platform built as a DBMS group project. Track meals, log workouts, and book expert consultations — all in one place.

Stack: Node.js · Express.js · PostgreSQL (Supabase) · Vanilla JS · AI API · Vercel + Render

Modules

Food — Log meals, track macros, get AI diet plans
Exercise — Browse exercises, log sessions, AI workout routines
Consultant — Book appointments with AI pre-consult summaries
Auth — JWT authentication with role-based access control (User / Admin)
